## 회고

- 타입스크립트와 리액트를 함께 사용하는 게 아직 조금 헷갈린다. 인터페이스와 타입을 만들고 Props를 적재적소에 가져다 사용하는 매커니즘에 좀 더 익숙해져야 할 것 같다. 아이템 리스트 렌더링을 구현하는 부분에서 좀 헤맸다.
- 기본 css를 사용하다 테일윈드를 사용하니 정말 편리하다는 걸 느낀다.
- 토탈을 내는 푸터 부분을 아직 구현하지 못했다.
